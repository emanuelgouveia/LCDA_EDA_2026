# 3. Univariate Analysis

## Overview

This module focuses on analyzing individual variables in isolation to understand their distributions, patterns, and characteristics.

## Learning Objectives

- Analyze distributions of individual variables
- Create and interpret univariate visualizations
- Identify patterns, trends, and anomalies in single variables
- Calculate and interpret statistical measures for individual variables
- Understand different types of distributions

## Tasks

1. **Numeric Variable Analysis**
   - Create histograms to visualize distributions
   - Generate box plots to identify outliers and quartiles
   - Create density plots for smooth distribution visualization
   - Calculate skewness and kurtosis
   - Test for normality (e.g., Shapiro-Wilk test)

2. **Categorical Variable Analysis**
   - Create frequency tables
   - Generate bar charts and pie charts
   - Calculate proportions and percentages
   - Identify dominant categories
   - Detect rare categories

3. **Distribution Analysis**
   - Identify distribution types (normal, uniform, exponential, etc.)
   - Compare empirical distributions to theoretical ones
   - Analyze distribution parameters
   - Identify multimodal distributions

4. **Central Tendency and Dispersion**
   - Calculate and compare mean, median, and mode
   - Measure spread using variance, standard deviation, and IQR
   - Identify when mean vs. median is more appropriate
   - Analyze coefficient of variation

5. **Visualization Best Practices**
   - Choose appropriate plot types for different variable types
   - Add proper labels, titles, and legends
   - Use color and style effectively
   - Create publication-quality figures
